⚠️ One-Time Setup — Do This Once!

  1. Go to jsonbin.io on your phone or computer and create a free account
  2. After login, click "New Bin", type {"products":[]} inside, then click Create
  3. Copy the long Bin ID shown (e.g. 66a1b2c3d4e5f67890abcdef)
  4. Go to Account → API Keys, click "Create API Key", copy it
  5. Open this HTML file in Notepad, find the two lines at the top of the script that say JSONBIN_KEY and BIN_ID, paste your values inside the quotes, save the file
  6. Upload the updated file to GitHub as index.html — done! ✅
